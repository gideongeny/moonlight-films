# Android App Screenshots

This folder contains screenshots of the StreamLux Android app taken from a mobile device.

## Required Images

Please add the following screenshots here:

1. **movies-section.png** - Movies section showing F1 movie, Popular section with Zootopia 2 and The Shadow's Edge
2. **tv-shows-section.png** - TV Shows section showing "Heated Rivalry" and Popular section with Stranger Things and IT: Welcome to Derry
3. **search-page.png** - Search page with anime girl illustration and search bar
4. **sign-in-page.png** - Sign In page with Google/Facebook login and email/password fields

## How to Add Images

1. Take screenshots from your Android phone
2. Transfer them to your computer
3. Rename them to match the names above
4. Place them in this folder
5. Commit and push to GitHub

The images will then be available at:
`https://raw.githubusercontent.com/gideongeny/STREAMLUX/main/Images/Android/[filename].png`
